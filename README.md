シンプルで使いやすい牢獄プラグインです。

罪を犯したプレイヤーをBANせずに晒し者にするには最適なプラグインです。


日本語にのみ対応しています / Japanese Only! Sorry ^^;


# 基本的な機能

✅ 監獄の座標と半径を設定可能

✅ 刑期が過ぎると自動的に設定した座標に釈放されます

✅ オフラインのプレイヤーであってもスポーン後にテレポートされます

✅ 指定された半径を超えたら戻されます

✅ IPベースのサブアカウント対策

✅ 投獄者への定期的なメッセージ

✅ 見やすい投獄者一覧表



IPを元にサブアカウントを判定し、自動的に投獄します。

オフラインモードのサーバーでも安心です。

# コマンド一覧

/setjail <半径>  自分がいる座標を中心に、指定した半径で牢屋の範囲を設定します

/setrelease  自分がいる場所を釈放時にテレポートされる場所に設定します

/sethologram  投獄されている人の一覧表をスポーンさせます

/jail <プレイヤー名> <刑期> <理由>  投獄するコマンド

/unjail <プレイヤー名>  釈放するコマンド
